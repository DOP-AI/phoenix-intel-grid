# Anomaly Detection Framework

## Mission

Phoenix Intel Grid uses anomaly detection to identify instability patterns before uncontrolled escalation produces widespread human, environmental, or infrastructure harm.

Built under the DOP-AI doctrine:

> Dignity Over Power

Anomaly detection exists to improve operational awareness and resilience.

Not to automate opaque control systems.
